Yield function

\[f = \sqrt{J_2} - (A - B I_1)\]

\[g = \sqrt{J_2} + b I_1\]

Parameters

  • mohr_coulomb_type: choose from :circumscribed, :inscribed and :plane_strain
  • c: cohesion
  • ϕ: internal friction angle
  • ψ: dilatancy angle
  • tensioncutoff: set limit of mean stress or false
